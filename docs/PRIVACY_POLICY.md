# Privacy Policy

Last updated: June 13, 2026

RiskWise is an educational options-risk check app. This draft policy is intended for the TestFlight beta and should be reviewed before public App Store release.

## Information RiskWise Uses

RiskWise may store account information, profile preferences, risk rules, saved checks, chat history, uploaded-file metadata, and user-confirmed contract details. These are used to personalize risk explanations and keep the app useful across sessions.

RiskWise does not need brokerage credentials for the current beta. If broker connection features are added later, they should use a separate consent flow and a separate privacy review.

## Market And AI Data

RiskWise may send non-sensitive trade-check context to the backend so it can calculate risk metrics and generate educational explanations. If hosted AI providers are configured, the backend may send compact prompts containing the user question, selected check, and relevant profile preferences. The app should not send passwords, payment details, or unnecessary personal identifiers to AI providers.

## Uploads

If the user uploads a screenshot or file, RiskWise stores only the metadata and extracted/confirmed trade context needed for analysis unless a future version explicitly asks to store the original file.

## Data Deletion

Users should be able to request deletion of their RiskWise account data, including saved checks, chat history, profile settings, and uploaded context metadata.

## Educational Use

RiskWise is educational decision-support software. It does not execute trades, provide financial advice, or guarantee investment outcomes.
